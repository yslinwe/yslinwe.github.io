#### 命令行

查看可用存储设备、磁盘分区等相关信息，找到你的存储设备名称

```
lsblk
```

![](https://pic.imgdb.cn/item/64fc20e3661c6c8e54ee4823.jpg)

```
mkdir /mnt/disk
```

```
mount /dev/mmcblk0p1 /mnt/disk
```

```
cd /mnt/disk
```

```
ls
```
