### 题目描述

大家都知道斐波那契数列，现在要求输入一个整数n，请你输出斐波那契数列的第n项（从0开始，第0项为0，第1项是1）。 

n≤39*n*≤39

### 示例1

#### 输入

```
4
```

#### 返回值

```
3
```

```c++
class Solution {
public:
    int Fibonacci(int n) {
        if(n<2)
            return n;
       return Fibonacci(n-1)+Fibonacci(n-2);
    }
};
```
