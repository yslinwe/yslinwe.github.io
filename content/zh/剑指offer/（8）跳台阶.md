### 题目描述

一只青蛙一次可以跳上1级台阶，也可以跳上2级。求该青蛙跳上一个n级的台阶总共有多少种跳法（先后次序不同算不同的结果）。

```c++
class Solution {
public:
    int jumpFloor(int number) {
        if(number<=2)
            return number;
        int first = 1;
        int second = 2;
        int result = 0;
        for(int i = 2;i<number;i++)
        {
            result = first + second;
            first = second;
            second = result;
        }
        return result;
    }
};
```
